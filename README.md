# Estudos de Python

Este repositório reúne meus exercícios, desafios e pequenos projetos enquanto construo minha base em programação com Python.

## Organização

- `01_fundamentos/` — exercícios separados por assunto
- `02_desafios/` — exercícios que combinam mais de um conceito
- `03_mini_projetos/` — projetos pequenos para aplicar o que aprendi

## Conteúdos já praticados

- Entrada e saída de dados
- Operações matemáticas básicas
- Condicionais com `if`, `elif` e `else`
